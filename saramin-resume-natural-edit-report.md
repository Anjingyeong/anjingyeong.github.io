# Saramin Resume Natural Edit Report

## 수정 방향

- AI 탐지 회피용 문장이 아니라, 실제 프로젝트에서 한 일이 자연스럽게 보이도록 다듬었습니다.
- 추상적인 역량 표현보다 모델, 데이터, 파이프라인, 오류 확인, 개선 항목이 드러나는 문장을 우선했습니다.
- 스마트 안전 관제 시스템은 대표 프로젝트로 유지하되, 확인되지 않은 정량 성능을 넣지 않았습니다.
- LLM Wiki/RAG는 보조 대표 프로젝트로 두고, 실제 구현이 불확실한 고급 검색 항목은 개선 방향으로 표현했습니다.
- 사람인 입력란에 붙이기 쉽도록 bullet을 1~2줄 안에서 읽히게 줄였습니다.

## 바로 쓸 수 있는 짧은 대체 문장

- YOLO Pose와 LSTM을 활용해 CCTV 영상의 이상행동 판단 구조를 설계했습니다.
- RTSP 입력, AI 추론, MQTT 이벤트, WebSocket 알림을 분리해 실시간 관제 흐름을 정리했습니다.
- FP/FN 사례를 기준으로 hard negative 재학습 후보와 threshold 개선 항목을 분리했습니다.
- 의료 영상의 왜곡과 조명 변화를 반영하기 위해 Grid Distortion과 Elastic Deform을 적용했습니다.
- 프로젝트 기록과 오류 해결 과정을 Evidence Wiki로 정리해 다시 확인할 수 있는 근거 자료로 만들었습니다.
