### 4. VAE 기반 비지도 학습 유방암 병변 검출 시스템

- **기간:** 2024.03 ~ 2024.10
- **형태:** 팀 프로젝트
- **역할:** 이상 탐지 파이프라인 설계, 커스텀 손실 함수 구현, 재구성 오차 기반 후처리
- **기술:** Python, TensorFlow, VAE (Variational AutoEncoder), 차영상 오차 함수
- **성과:** 산업통상자원부 장관 주관 공학혁신상 수상

#### Situation

유방 초음파 영상은 정밀한 병변 주석(Labeling) 작업에 막대한 전문의 리소스와 비용이 발생하여, 신뢰도 높은 지도학습 데이터셋을 대량 확보하기 어려운 한계가 존재했습니다. 기존 라이브러리에 의존하는 블랙박스 구조의 모델은 암세포와 정상 조직 간의 미세한 픽셀 분포 차이를 섬세하게 포착하기 어려웠습니다.

#### Task

값비싼 라벨 확보의 병목을 해결하기 위해 정상 조직 데이터만 학습한 뒤 정상 패턴에서 벗어나는 영역을 아노말리(Anomaly)로 식별하는 비지도 이상 탐지 기반의 병변 후보 탐지 파이프라인을 구현하고자 했습니다.

#### Action

- **저수준 모델링 기반 학습 통제:** TensorFlow 저수준 API를 활용해 잠재 공간 분산 오차(KLD)와 이미지 재구성 오차(MSE)의 텐서 가중치를 조작하는 커스텀 손실 함수(Loss)를 밑단부터 설계하여 비지도 VAE 파이프라인을 구축했습니다.
- **동적 노이즈 마스킹 개발:** 입력 이미지와 디코딩된 복원 이미지 간의 차이를 계산한 Reconstruction Error Map을 생성하고, 노이즈 오탐을 제어하기 위해 픽셀 분포 비율을 실시간 분석하는 `Dynamic Threshold` 후처리 알고리즘을 단독 개발했습니다.

#### Result

- 라벨이 전혀 없는 학습 환경에서도 분할 알고리즘 정밀도를 Dice Coefficient 기준 상시 90% 수준으로 안정화시켰습니다.
- 노이즈가 심한 초음파 환경에서도 오탐율을 대폭 하향시키며 산업통상자원부 장관 주관 🏆 공학혁신상을 수상했습니다.

#### Learned

AI 모델 설계 시 라이브러리 블랙박스에 의존하는 대신 저수준 텐서 연산을 통제하여 특성에 맞는 커스텀 손실 함수를 설계해야 근본적인 정밀도를 얻을 수 있다는 점을 경험했습니다. 또한 실무 배포 시 사후 평가 검증 로직이 모델 자체 성능만큼 중요하다는 것을 배웠습니다.

### 3. RF-DETR 실시간 대장 내 용종 검출 시스템

- **기간:** 2023
- **형태:** 팀 프로젝트
- **역할:** 모델 실험, 데이터 증강, 파인튜닝 파이프라인 개선
- **기술:** Python, PyTorch, RF-DETR, DINOv2, OpenCV
- **성과:** 제17회 건양대학교 캡스톤디자인 경진대회 대상/금상, 전국 공학교육혁신 컨소시엄 창의적 종합설계 경진대회 동상

#### Situation

대장 내시경 영상에서 용종을 실시간으로 검출하는 프로젝트였습니다. 의료 영상은 조명 변화, 반사, 장벽의 움직임 때문에 일반 이미지보다 검출 난도가 높았습니다. 또한 실제 활용을 고려하면 정확도뿐 아니라 추론 속도도 중요했습니다.

초기에는 CenterNet과 RetinaNet의 장점을 결합한 커스텀 모델을 시도했지만, 제한된 데이터와 의료 영상 특성 때문에 기대한 정확도가 나오지 않았습니다.

#### Task

정확도 한계를 해결하면서도 실시간 추론 가능성을 유지하는 방향으로 모델링 전략을 바꿔야 했습니다.

- 커스텀 모델 실패 원인 분석하기
- 의료 영상에 적합한 객체 탐지 모델 선택하기
- 부족한 라벨링 데이터를 보완할 증강 전략 적용하기
- 실시간 추론이 가능한 수준으로 파이프라인 최적화하기

#### Action

- 초기 커스텀 모델 실험 결과를 분석한 뒤, 구조를 새로 만드는 방식보다 검증된 객체 탐지 모델을 의료 데이터에 맞게 파인튜닝하는 방향으로 전환했습니다.
- RF-DETR 기반 파인튜닝 파이프라인을 구성하고, 대장 내시경 영상의 특성을 반영한 데이터 증강을 적용했습니다.
- 단순 회전/반전뿐 아니라 Grid Distortion, Elastic Deform 계열의 증강을 사용해 장벽 왜곡과 점막 형태 변화를 모델이 학습할 수 있도록 했습니다.
- OpenCV 기반 전처리와 추론 결과 확인 과정을 반복하며, 오탐과 미탐 케이스를 비교했습니다.

#### Result

- 기존 베이스라인 대비 mAP를 약 7%p 개선했습니다.
- 실시간 추론 가능성을 확인했고, 프로젝트 결과로 교내 캡스톤디자인 경진대회 대상/금상과 전국 경진대회 동상을 수상했습니다.
- 모델 구조 자체를 새로 만드는 것보다, 문제 상황에 맞는 모델 선택과 데이터 중심 개선이 더 효과적일 수 있다는 점을 배웠습니다.

#### Learned

처음 시도한 모델이 실패했을 때, 단순히 더 복잡한 모델을 만드는 방향으로 가지 않고 문제를 다시 정의한 것이 중요했습니다. 의료 영상에서는 모델의 새로움보다 데이터 특성과 평가 기준에 맞는 안정적인 개선이 더 중요하다는 점을 배웠습니다.

### 4. VAE 기반 비지도 학습 유방암 병변 검출 시스템

- **기간:** 2022
- **형태:** 팀 프로젝트
- **역할:** VAE 기반 이상 탐지 파이프라인 설계, 재구성 오차 기반 병변 후보 검출
- **기술:** Python, TensorFlow, VAE, Computer Vision
- **성과:** 산업통상자원부 장관 주관 공학혁신상 수상

#### Situation

초음파 영상에서 유방암 병변을 검출하는 프로젝트였습니다. 의료 데이터는 라벨링 비용이 높고, 모든 병변 케이스를 충분히 확보하기 어렵다는 문제가 있었습니다.

정답 라벨이 충분하지 않은 상황에서 병변 후보 영역을 찾기 위해, 정상 조직 이미지를 기준으로 이상 패턴을 탐지하는 비지도 학습 접근을 선택했습니다.

#### Task

라벨이 부족한 환경에서도 병변 후보 영역을 찾을 수 있는 파이프라인을 구현해야 했습니다.

- 정상 조직 이미지를 학습하는 VAE 모델 구성하기
- 입력 영상과 재구성 영상의 차이를 이용해 이상 후보 영역 찾기
- 노이즈를 줄이기 위한 임계값 처리와 후처리 로직 만들기
- 결과를 시각적으로 확인할 수 있는 형태로 출력하기

#### Action

- TensorFlow 기반 VAE 모델을 구성해 정상 조직 이미지의 분포를 학습하도록 했습니다.
- 입력 이미지와 모델이 재구성한 이미지의 차이를 계산해 Reconstruction Error Map을 만들었습니다.
- 단순 고정 임계값만으로는 노이즈가 많이 남는 문제가 있어, 이미지 내 픽셀 분포를 기준으로 동적 임계값을 조정하는 후처리 로직을 적용했습니다.
- 병변 후보 영역을 시각화하고, 결과 이미지를 비교하며 오탐 영역을 줄이는 방향으로 반복 개선했습니다.

#### Result

- 라벨이 부족한 의료 영상 환경에서 비지도 학습 기반 이상 탐지 가능성을 확인했습니다.
- Dice Coefficient 기준 약 90% 수준의 분할 성능을 목표로 파이프라인을 개선했습니다.
- 프로젝트 결과로 산업통상자원부 장관 주관 공학혁신상을 수상했습니다.

#### Learned

이 프로젝트를 통해 AI 모델 성능은 모델 구조만으로 결정되지 않고, 데이터 전처리와 후처리 기준에 크게 영향을 받는다는 점을 배웠습니다. 특히 의료 영상에서는 작은 노이즈도 결과 해석에 영향을 주기 때문에, 결과를 시각적으로 확인하고 임계값을 조정하는 과정이 중요했습니다.
